## 安装步骤

```
git clone https://github.com/ni-ning/rest_framwork.git

cd rest_framwork
python3 -m venv env

pip install django
pip install djangorestframework
pip install coreapi

# 数据库 mysql 时
pip install mysqlclient
```


## setup

```
python manage.py runserver
```
